# 👤 Profile Card Moderno em HTML e CSS

Projeto desenvolvido em **HTML5** e **CSS3** com o objetivo de criar um **Card de Perfil moderno e responsivo**, apresentando foto do usuário, nome, profissão, ícones de redes sociais e um botão de ação com um design elegante e minimalista.

Este projeto é ideal para portfólios, páginas pessoais, equipes de desenvolvimento e interfaces modernas.

---

## 📸 Prévia

O projeto exibe um cartão de perfil centralizado sobre um fundo com degradê em tons de azul. O card contém uma imagem de destaque, informações do usuário, ícones de redes sociais e um botão estilizado na parte inferior.

---

## ✨ Funcionalidades

- 👤 Card de perfil moderno
- 🖼️ Foto do usuário em destaque
- 📝 Nome e profissão personalizados
- 🌐 Ícones para redes sociais
- 🎨 Fundo com efeito degradê
- 🔘 Botão de ação estilizado
- 📱 Layout responsivo
- ⚡ Desenvolvido apenas com HTML e CSS
